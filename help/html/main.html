Guía de ayuda de la Aplicación

Introducción

Esta aplicación te permite gestionar naves, tripulantes y misiones de la flota espacial. Puedes visualizar, añadir, eliminar y modificar elementos en cada una de las secciones.

Además, podrás elegir el modo oscuro para reducir la fatiga visual o el modo claro para una experiencia de usuario más clara.
